# SEO Toolkit

A comprehensive SEO analysis toolkit for metadata analysis and Lighthouse score tracking.

## Features

- **Metadata Analyzer**: Upload or link to XML sitemaps, extract metadata from all pages, bulk edit, and export to CSV
- **Lighthouse Dashboard**: Run Lighthouse audits, track scores over time, and visualize trends

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Deployment

This project is configured for Netlify deployment. Simply connect your GitHub repository to Netlify and it will build automatically.
